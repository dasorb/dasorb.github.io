查看网络连接是否正常

使用 ping 测试网络连接是否通常

如果没有网络IP，请检查虚拟机网络编辑器NAT设置是否分配IP。

否则请给机器设置静态IP：

etc/network/interfaces

安装SSH服务器通信软件

登陆root用户

设置root用户密码 sudo passwd root
启动root用户 su root

安装ssh软件

安装命令 apt-get install openssh-server

启动服务 service ssh start

查看进程 ps -e |grep ssh

配置ssh使其允许远程连接

打开文件：/etc/ssh/sshd_config
修改代码：PermitRootLogin yes
重启ssh服务：service ssh restart

使用putty.exe程序在windows上打开session会话输入地址远程打开LINUX终端

安装hadoop

介绍

高可用 （HA）+ 负载均衡 = 多服务器镜像+请求平均分配

官方下载：hadoop.apache.org

官方文档：hadoop.apache.org/docs/r1.0.4/cn/mapred_tutorial.html

安装方式：

  1. 本地模式：用于测试，直接解压就能运行
  2. 伪分布模式：用于开发，必须进行网络连接
  3. 分布模式：用于生产
